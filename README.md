## 关于这个项目

StartAI.fun 是一个 AI 工具导航站，帮助零基础用户从零开始掌握 AI 工具。

采用**五步学习路径**设计：

| 步骤 | 内容 | 说明 |
|:---:|------|------|
| ① | 领养 | 入门 AI 工具推荐 |
| ② | 喂养 | API 与数据源配置 |
| ③ | 教技能 | Prompt 写作与训练 |
| ④ | 配装备 | MCP 等扩展工具 |
| ⑤ | 找伙伴 | Multi-Agent 协作 |

## 技术栈

- 纯 HTML/CSS/JavaScript，无框架依赖
- 响应式设计，支持移动端
- SEO 优化（sitemap、JSON-LD、robots.txt）
